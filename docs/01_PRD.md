# 01 PRD

## Product Vision

Scientific Spaces AI Learning OS helps users learn from Scientific Spaces blog content, connected papers, knowledge graphs, and grounded AI tutoring workflows.

## Users

- Learners who want structured explanations.
- Researchers who want source-grounded discovery.
- Maintainers who need a clear content and AI pipeline.

## Bootstrap Scope

Bootstrap creates documentation, directory structure, milestones, and project status.

Bootstrap does not implement product features.

## Product Capabilities

- Source pipeline for Scientific Spaces blog content.
- Reader system for articles.
- Grounded RAG assistant with citations.
- Learning management and history.
- Zotero paper integration.
- Knowledge graph linking concepts, theories, papers, and experiments.
- AI research tutor modes.

## Non-Goals

- No backend implementation in Bootstrap.
- No frontend implementation in Bootstrap.
- No crawler implementation in Bootstrap.
- No RAG implementation in Bootstrap.
- No business logic in Bootstrap.
