# 10 UI Spec

## Purpose

Define future user interface surfaces.

## Planned Views

- Dashboard
- Article List
- Article Detail
- Search Results
- Reading History
- Knowledge Graph View
- AI Tutor View

## User Flow

1. Open website.
2. Search article or concept.
3. Open article.
4. Read content.
5. Ask AI tutor questions.
6. Review sources and learning history.

## Bootstrap Constraint

No frontend application or UI implementation is included in Bootstrap.
