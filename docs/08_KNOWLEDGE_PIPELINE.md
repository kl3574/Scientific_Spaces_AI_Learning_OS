# 08 Knowledge Pipeline

## Purpose

Define the future path from source content to structured learning knowledge.

## Planned Flow

1. Discover Scientific Spaces blog articles.
2. Parse article structure and metadata.
3. Convert content to Markdown.
4. Store article records.
5. Chunk documents by Markdown structure.
6. Generate embeddings.
7. Link concepts, papers, and experiments.
8. Serve grounded answers with citations.

## Bootstrap Constraint

This is a planning document only. No crawler, parser, converter, embedding, or graph pipeline is implemented in Bootstrap.
